## пояснитель: моментальный ии помощник для платформы LXP IThub
# описание проекта 
данный проект поможет студентам и преподавателям в моментальном разборе непонятного слова
# установка и запуск
Туториал :3 
1) заходим в приложение или сайт LXP IThub
2) регестрируемся или входим в свой аккаунт
3) нажимаем на "дисциплины"
4) после выбора дисциплины мы должны выбрать нужную тему
5) при находе непонятного слова выделяем его курсором мышки
6) нажимаем на правую кнопку мыши и выбираем "почему так?"
7) после этого вас кидает на сайт, где подробно рассказывают про это слово

# технические требования
Объяснение выделенного кода на LXP/GitHub
 1 Объяснение ошибок компиляции/выполнения
 2 Генерация примеров кода по запросу
 3 Поддержка 15+ языков программирования
 4  Русский/английский интерфейс

Архитектурные
 1 Frontend: Браузерное расширение (Chrome/Firefox)
 2 Backend: REST API (Python/FastAPI)
 3 AI: Интеграция с LLM API (OpenAI/Anthropic + локальные модели)
 4 Кэширование: Redis для повторяющихся запросов
 5 Хранилище: PostgreSQL для истории запросов (опционально)

Производительность
 1 Время ответа < 3 секунд
 2 Поддержка 1000+ concurrent пользователей
 3 Кэширование частых запросов
 4 Rate limiting (10 запросов/минута на пользователя)

Безопасность
 1 Аутентификация через LXP/GitHub OAuth
 2 Валидация входных данных
 3 Защита от инъекций в промпты
 4 Шифрование конфиденциальных данных

Интеграции
 1 LXP API для контекста обучения
 2 GitHub API для анализа кода
 3 OpenAI API, Anthropic Claude, локальные LLM
 4 Система мониторинга (Prometheus + Grafana)

UX
 1 Минимальный UI (popup/sidebar)
 2 Горячие клавиши для быстрого доступа
 3 История предыдущих объяснений
 4 Настройка уровня детализации

# процесс тестирования

1. Виды тестирования
Модульное: тестирование отдельных функций и сервисов
Интеграционное: взаимодействие между компонентами, API, БД
E2E: полные пользовательские сценарии в браузере
Нагрузочное: производительность при высокой нагрузке
Безопасность: проверка уязвимостей и защита данных

2. Ключевые сценарии
 Объяснение выделенного кода на LXP/GitHub
 Анализ ошибок компиляции и выполнения
 Генерация примеров кода
 Работа с разными языками программирования
 Интерфейс расширения браузера

3. Инфраструктура
 Автоматические тесты в CI/CD пайплайне
 Тестовые среды (staging)
 Mock внешних сервисов (LLM API)
 Тесты для разных браузеров

4. Критерии качества
 90% покрытие кода тестами
 Все тесты проходят перед релизом
 Время ответа < 3 секунд
